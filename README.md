# spicywar.io
A multiplayer game where you try to become the spiciest of them all!

## Setup & Running
### Debug
#### Building
```sh
npm install --include=dev
npm run build
```
#### Staring
Once the game is built, run `npm start` to execute it.
### Production 
#### Building
```sh
npm install --include=dev
npm run prod
```
#### Starting
`npm start`
